Hello friends,
This is a new css grid layout by which you can easily complete your web design task very fast. It's very lightweight so your website load very faster with this framework.
It's still updated with new components and features always. It has built in responsive features so you don't worry about responsive coding.

Basic Usage Instructions
1. Download arb framework package. unzip it to your desired folder and follow the steps to get started.

2. add <link rel="stylesheet" href="css/arb_grid.css"> before tag.

3. add <script src="js/arb_min.js"></script> before body tag. ( Make sure you call latest jquery before calling js/arb_min.js )

4. Basic layout style: 
<div class="container">
      <div class="row">
            <div class="col-12">
                  Here your codes
            </div>
      </div>
</div>
5. Navigation: 
<nav class="navbar">   
    <ul class="nav-menu">
        <li><a href="#home">Home</a></li>
            <li><a href="#sections">Basics</a>
                <ul class="sub-menu">
                        <li><a href="#item-1">Adding files</a></li>
                <ul>
            </li>
    </ul>
</nav>
Notes: Use .nav-menu class for navigation menu list and #collapse id for responsive menu link. you can also use .sub-menu class to add dropdown menu and .navbar-fixed class to fixed menu to your navigation list.

Grid systems
Arb css grid is responsible and it is useful for faster website development. It comes with 12 grid/column and every grid is caled by col-x prefix.

Example: For 12 column we call it col-12 and for 6 columns we will call col-6. Here are some live columns given below.
